Module matrix

matrix math library (mat2, mat3, mat4) optimized for luajit with ffi, allocation sinking, ieee 754 hex floats uses row-major storage for cache efficiency and simd-friendly sequential access matrices stored as contiguous arrays of rows for optimal memory locality
