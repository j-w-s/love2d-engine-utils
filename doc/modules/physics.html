add (id, x, y, w, h, type, active, mask, on_collide)
add or update an aabb collider in the world

Parameters:

  • id string|number unique id
  • x number
  • y number
  • w number
  • h number
  • type string (optional) e.g., 'solid', 'trigger', 'player'
  • active boolean (optional) if false, is ignored by queries
  • mask number (optional) collision bitmask (default: 0xFFFFFFFF)
  • on_collide function (optional) callback(id, other_id, other_col, nx, ny)

add or update a polygon collider in the world vertices are specified in world space

Parameters:

  • id string|number unique id
  • x number reference x position
  • y number reference y position

  • type string (optional)

  • mask number (optional)
  • on_collide function (optional)

move (id, dx, dy, use_swept, stop_on_first)
move a collider with collision resolution

Parameters:

  • id string|number id to move
  • dx number
  • dy number
  • use_swept boolean (optional) use swept aabb for continuous collision
  • stop_on_first boolean (optional) stop on first collision (faster)

Returns:

  1. number new_x
  2. number new_y
  3. table hits { x = {id=col}, y = {id=col} }

raycast through the world

Parameters:

  • x1 number start x
  • y1 number start y
  • x2 number end x
  • y2 number end y
  • type_filter string (optional) only check this type
  • mask_filter number (optional) collision mask filter
  • callback function (optional) fn(id, col, hit_x, hit_y) -> bool (return true to stop)

Returns:

    table array of hits: { {id, col, x, y, distance}, ... }
